# CryptoCore
### Diploma project

Проста у використанні, але криптографічно надійна програма для локального шифрування файлів. Спроєктована з акцентом на швидкодію та безпечне управління оперативною пам'яттю.
Підтримує класичні блокові алгоритми AES-256, Blowfish, 3DES та DES у режимі зчеплення блоків (CBC) і з вирівнюванням за стандартом PKCS#7.
Для некритичних завдань доступний альтернативний потоковий шифр, що використовує векторні SSE-інструкції процесора

### Для правильної роботи програми на комп'ютері мають бути встановлені:
* **Python 3** (для запуску графічного інтерфейсу).

### Як запустити:
1. Завантажте та **розпакуйте архів** із програмою у зручну папку.
2. Запустіть головний файл програми **`main.py`** (через командний рядок або подвійним кліком).
